# GainForest Scorer Visualizer

Simple React/Vite app for browsing scored GraphQL records.

## Run

```bash
npm install
npm run dev
```

The app uses `https://subarid-diapasonal-bibi.ngrok-free.dev/graphql` by default.
Override with:

```bash
VITE_GRAPHQL_URL=https://your-host/graphql npm run dev
```
